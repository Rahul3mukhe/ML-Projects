# Amazon Alexa Review Sentiment Analysis

This project focuses on analyzing customer sentiment from Amazon Alexa product reviews using Natural Language Processing (NLP) and machine learning techniques.

---

##  Project Overview
- Performs sentiment classification on customer reviews
- Uses text preprocessing and feature extraction techniques
- Evaluates model performance using accuracy and cross-validation

---

##  Dataset Description
- Amazon Alexa customer review dataset
- Reviews labeled for sentiment classification

---

##  Methodology
1. Data loading and inspection  
2. Text preprocessing:
   - Stopword removal
   - Stemming
   - Tokenization
3. Feature extraction using CountVectorizer  
4. Model training and evaluation  
5. Cross-validation for robustness testing  

---

##  Model Used
- Random Forest Classifier  

---

##  Model Performance (From Notebook Output)
- Training Accuracy: **99.45%**
- Testing Accuracy: **94.39%**
- 10-Fold Cross-Validation Mean Accuracy: **93.19%**
- Variance: **0.01198**
